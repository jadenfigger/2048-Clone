How the game works:

It is played on a 4x4 grid using the directional arrows keys on the computer. Use the directional arrows on a computer to move tiles of different colours and values with a scrolling animation. When two tiles of the same number are side by side, they can merge and form a new tile of double value (e.g. two tiles of value « 2 » give a tile of value « 4 »).

How the AI works:

The AI plays the game several times using completely random moves! The average final score per starting position is calculated. The shot with the highest score is chosen. By default, the AI plays 20 games per move. Increase the number of moves to make AI more efficient and strong. It is interesting to note that even though AI is based on random moves, it plays quite well.
